# Polices de marque Île-de-France Mobilités

La charte IDFM impose deux familles **propriétaires** (non redistribuables, donc
absentes du dépôt). Déposez ici les fichiers `.woff2` fournis par la médiathèque
IDFM ; les `@font-face` de `src/app/globals.css` les chargent automatiquement.

| Usage (charte) | Famille | Fichiers attendus |
|---|---|---|
| Titres / espaces contraints (display, UI, CTA) | **IDF Voyageur** | `IDFVoyageur-Regular.woff2`, `IDFVoyageur-Bold.woff2` |
| Textes courants / espaces denses (corps) | **Grand Paris** | `GrandParis-Regular.woff2`, `GrandParis-Medium.woff2`, `GrandParis-Bold.woff2` |

Tant que les fichiers ne sont pas présents, un fallback institutionnel
sans-serif (Geist / système) s'applique sans casser la mise en page.

Source officielle : médiathèque Île-de-France Mobilités / plateforme PRIM
(Prescriptions signalétiques).
